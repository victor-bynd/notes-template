Example dashboard

Worked example using tabs, cards, lists, banner, dialog, toast, table, and an embedded Mermaid diagram. Delete _examples/ when initialising a project.

Adding a prototype

  1. Duplicate _template.html with a kebab-case filename.
  2. Add an entry to prototypes.md.
  3. Add a card to this index matching that entry.
  4. Build inside the <main> using classes from SYSTEM.md.
